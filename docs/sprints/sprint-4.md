# Sprint 4 - Analytics

## Goal
Make users feel progress and insight.

## Epic 4.1: Data Tracking

### Story 4.1.1 - Session Model
**Description:**
Define DB schema for sessions.

**Acceptance Criteria:**
- Stores:
  - duration
  - type
  - date

## Epic 4.2: Stats Dashboard

### Story 4.2.1 - Stats API
**Description:**
Fetch aggregated data.

**Acceptance Criteria:**
- Returns:
  - total time
  - avg session
  - sessions count

### Story 4.2.2 - Stats UI
**Description:**
Display analytics dashboard.

**Acceptance Criteria:**
- Shows:
  - daily focus time
  - weekly trend

## Epic 4.3: Insights

### Story 4.3.1 - Insight Engine
**Description:**
Generate basic insights.

**Acceptance Criteria:**
- Examples:
  - "Best focus time"
  - "Avg session length"
